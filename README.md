# ETL_Process
Creation of a SQL database through the etl process (extraction, transformation and loading) from a csv file.

**[ES]**

La base de datos que se utlizará inicialmente es *https://raw.githubusercontent.com/diljeet1994/Python_Tutorials/master/Projects/Advanced%20ETL/crypto-markets.csv*. 
Se manipulará con el uso de la libreria **pandas**, donde extraeremos los datos, los transformaremos para usar con SQL y crearemos una *tabla.db* para visualizar el resultado.

Para visualizar de mejor manera y manipular usando comandos SQL, podemos utilizar SQLiteBrowser, el enlace de descarga para Windows es *https://sqlitebrowser.org/dl/*, para MacOs es *https://www.macupdate.com/app/mac/38584/db-browser-for-sqlite*.

**[EN]**

The database that will be used initially is *https://raw.githubusercontent.com/diljeet1994/Python_Tutorials/master/Projects/Advanced%20ETL/crypto-markets.csv*.
It will be manipulated with the use of the **pandas** library, where we will extract the data, transform it to use with SQL and create a *table.db* to visualize the result.

To better visualize and manipulate using SQL commands, we can use SQLiteBrowser, the download link for Windows is *https://sqlitebrowser.org/dl/*, for MacOs it is *https://www.macupdate.com/app/mac/38584/db-browser-for-sqlite*.
